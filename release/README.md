# E2E refined dataset
This is a refined dataset of the [E2E dataset](https://github.com/tuetschek/e2e-dataset/releases/download/v1.0.0/e2e-dataset.zip).

_Authors: Anonymous_

## Files
- training data
  - e2e_train.json
- validation data
  - e2e_valid.json
- test data
  - e2e_test.json
- reordered test data
    - e2e_test_ex_0.json
